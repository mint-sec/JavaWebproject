# 第 1 周前端原型草图

本周前端页面以“单车监控演示”为核心，先跑通假数据展示链路。

## 页面结构

```text
+----------------------------------------------------------------------------------+
| 顶部标题区：项目名称 / 当前车辆 / 系统时间 / 监控场景                             |
+----------------------------------------------------------------------------------+
| 概览卡片：当前温度 | 车厢湿度 | 配送速度 | 风险等级                               |
+----------------------------------------------------------------------------------+
| 车辆地图区域                 | 温度折线图区域                | 告警信息面板      |
| - 配送中心与配送点           | - 最近 30 分钟温度趋势        | - 最新告警卡片     |
| - 当前车辆位置               | - 安全温区标识               | - 风险等级标签     |
| - 剩余路程与调度建议         | - 当前温度与趋势说明         | - 处置建议文案     |
+----------------------------------------------------------------------------------+
```

## 第 1 周范围

1. 用静态页面先完成大屏布局，不依赖后端接口。
2. 用假数据演示一辆疫苗冷链车的温度变化、位置变化和告警变化。
3. 页面先突出三块核心区域：
   - 车辆地图区域
   - 温度折线图区域
   - 告警信息面板

## 第 2 周可继续扩展

1. 把假数据替换成后端接口。
2. 增加车辆状态卡片联动刷新。
3. 接入真实地图组件和轨迹回放。
